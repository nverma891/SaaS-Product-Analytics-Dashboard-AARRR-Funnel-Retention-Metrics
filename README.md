# SaaS-Product-Analytics-Dashboard-AARRR-Funnel-Retention-Metrics
A simulated SaaS product analytics dashboard built to track the full AARRR funnel (Acquisition, Activation, Retention, Referral, Revenue) across a 500-user dataset. The project demonstrates end-to-end product analytics thinking — from raw data modeling to executive-ready recommendations.
The core insight surfaced through cohort analysis: one onboarding variant drove 22% higher Day-7 retention than the control, providing a clear, data-backed roadmap for action.
